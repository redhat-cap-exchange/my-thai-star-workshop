# "Road to cloud-native" Workshop

The "Road to cloud-native" workshop is a 1-day event with a series of hands-on-labs which are designed to familiarize participants with cloud-native concepts and give them a taste of using OpenShift for building and managing cloud-native applications.

## Workshop Outline

* [Setup the development environment](docs/setup.md)
* [Create a workspace](docs/workspace.md)
* [Lab 1 - A first build](docs/lab1.md)
* [Lab 2 - Source-to-Image (S2I) Build](docs/lab2.md)
* [Lab 3 - Builds and Deployments using Templates](docs/lab3.md)
* [Lab 4 - Customizing the S2I build process](docs/lab4.md)
* [Lab 5 - Simple CI/CD ](docs/lab5.md)
* [Lab 6 - Monitoring Application Health ](docs/lab6.md)
